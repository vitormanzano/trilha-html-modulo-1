História dos computadores

(Voltar)

Os primeiros computadores foram criados com o objetivo de fazer cálculos, como o Mark 1 sendo da 1a geração, os computadores foram até a 5a geração que é onde dizem que estamos, composta pela: IA, criptomoedas.

O que são Clients?

(Voltar)

Clientes é quando se "consome" a informação de algum lugar. Como quando alguém acessa uma pagina pelo notebook ou celular, computador. Ele se comunica através da internet com um outro computador, onde ele faz a requisição para entrar no site, e o server responde com o site.

O que são Servers?

(Voltar)

É onde fica armazenado alguns arquivos, que faz conexão para outros computadores, como se fosse um computador central. Servidores tendem a ser computadores mais potentes, para poder receber diversas requisições.

Como é uma estrutura básica de um HTML?

(Voltar)

O HTML(Hipertext Markup Language) é um hipertexto onde se tem tags, essas tags que são entendidas como páginas, cada tag quando lançada sobre um editor, tem uma determinada função. Algumas tags: h1,h2,...,h6,p,strong,ul,ol,li,a,i,br entre outras.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Listar ordenadas: São aquelas que se tem uma ordem, podemos ter um exemplo de um ranking, onde se tem a lista indo de 1 a 10, ou seja, uma lista ordenada. Listas não ordenadas: São aquelas que não tem ordem, que não necessitam de ordem. Por padrão no HMTL a lista não ordenada ela aparece com bolinhas na frente das palavras.
